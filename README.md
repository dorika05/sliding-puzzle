# sliding-puzzle

## Játék rövid leírása

A program egy sliding puzzle játékot valósít meg. A felhasználónak lehetősége van 3x3-as 4x4-es, vagy
5x5-ös táblát választani. Egy adott játékot bármikor újrakezdhet, illetve egy játékállást fájlba menthet,
és visszatölthet. 

## Futtatás
A projekt forráskódja a `src` mappában található.
